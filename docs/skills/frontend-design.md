# ADVICE FOR BUILDERS

- Ban Inter, Roboto, Arial, and Space Grotesk from your design system entirely.
- Choose one extreme aesthetic direction per project and execute with unwavering commitment.
- Use dominant color with sharp accents instead of safe, evenly-distributed palette choices.
- Implement one orchestrated page load animation rather than scattered random micro-interactions throughout.
- Create custom background atmospheres with gradients, noise, or textures instead of solids.
- Break grid layouts intentionally with asymmetry, overlap, and diagonal flow for memorability.
- Match code complexity to vision: elaborate maximalism needs elaborate code, minimalism needs precision.
- Test every design decision against "would AI generate this?" and pivot immediately.
- Build reusable aesthetic systems with CSS variables that enforce cohesive, distinctive design language.
- Prioritize scroll-triggered and hover animations that genuinely surprise rather than meet expectations.
- Study extreme aesthetic movements (brutalism, art deco, maximalism) for inspiration beyond defaults.
- Audit competitors' sites and deliberately choose opposite aesthetic directions for differentiation strategy.
- Invest in distinctive display font licenses that competitors won't use or access.
- Document your aesthetic direction in writing before coding to maintain intentionality throughout.
- Review finished work asking "what will users remember?" and amplify that element aggressively.
